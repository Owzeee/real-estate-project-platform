Immo Neuf

Production SEO Launch Checklist

Final production QA checklist for technical SEO, indexing, previews, and search setup.

1. Deployment

  • Deploy the latest version of the project from ~/Desktop/real-estate-platform.
  • Set the production environment variable: NEXT_PUBLIC_SITE_URL=https://immoneuf.ci.
  • Confirm the production domain is the only canonical domain used by the site.
  • If preview or staging domains exist, make sure they are not being indexed.

2. Production Technical Checks

Robots

  • Open https://immoneuf.ci/robots.txt.
  • Confirm public crawling is allowed.
  • Confirm /wishlist, /auth/, /admin/, and /developer/ are blocked.

Sitemap

  • Open https://immoneuf.ci/sitemap.xml.
  • Confirm the sitemap loads correctly.
  • Confirm public pages, project pages, unit pages, and developer pages appear.

Metadata Checks

Check these pages:

  • Home page
  • /projects
  • One project detail page
  • One unit detail page
  • One developer detail page
  • /abidjan
  • /abidjan/terrains
  • /abidjan/bureaux
  • /cote-divoire

On each page confirm title, meta description, canonical, Open Graph tags, and Twitter tags are correct.

3. Search Console

  • Add and verify the domain in Google Search Console.
  • Submit https://immoneuf.ci/sitemap.xml.

URL Inspection

  • https://immoneuf.ci/
  • https://immoneuf.ci/projects
  • One project page
  • One unit page
  • One developer page
  • https://immoneuf.ci/abidjan
  • https://immoneuf.ci/abidjan/terrains
  • https://immoneuf.ci/abidjan/bureaux
  • https://immoneuf.ci/cote-divoire

For each URL, confirm Google can access it, confirm the canonical is correct, and request indexing if needed.

4. Social Preview Checks

  • Test the home page.
  • Test one project page.
  • Test one unit page.
  • Test one developer page.

Confirm preview image, title, and description are correct.

5. Pages That Must Stay Noindex

  • /wishlist
  • /compare
  • /auth/*
  • /admin/*
  • /developer/*

6. Final Validation

  • Confirm no staging domain appears in canonical tags, sitemap, Open Graph URLs, or robots output.
  • Confirm the production domain is consistent everywhere.
  • Confirm public pages are indexable.
  • Confirm utility and private pages are not indexable.

7. Optional Recommended Tasks

  • Add the site to Bing Webmaster Tools.
  • Connect Google Search Console with GA4 if GA4 is used.
  • Monitor index coverage, impressions, clicks, CTR, top queries, and top landing pages.

8. Priority URL Set

  • Home
  • Projects listing
  • 1 featured project
  • 1 unit detail page
  • 1 developer profile
  • Abidjan landing page
  • Terrains landing page
  • Bureaux landing page
  • Côte d’Ivoire landing page

Sign-Off

  • [ ] Robots verified
  • [ ] Sitemap verified
  • [ ] Metadata verified
  • [ ] Search Console verified
  • [ ] Sitemap submitted
  • [ ] Key URLs inspected
  • [ ] Social previews verified
